一般参加ガイド

  • 年齢確認：身分証をご提示ください（不可例：学生証の画像のみ等）。
  • パンフ・入場：パンフ1冊＋リストバンド配布／再入場可。
  • 抽選：12:00に会場内で実施。抽選券は入場時配布。
  • 持ち物：小銭・エコバッグ・水分。会場は一方通行の導線です。

混雑時は入場制限を行う場合があります。スタッフの指示に従ってください。

R-18 作品の取り扱い

  • 年齢確認後に該当エリアへ入場可能。
  • 撮影・配信は不可。掲載ガイドラインをご確認ください。

サークル参加ガイド

  • 参加資格：オリジナル/二次創作の頒布が主目的であること。
  • ブース種別・料金：1ブース 5,000円（机1/椅子2/机上POP可）。
  • 当日フロー：搬入→見本誌確認→開場→頒布→撤収。
  • 見本誌：表紙/奥付/年齢区分の表記必須。

欠席・委託の連絡
